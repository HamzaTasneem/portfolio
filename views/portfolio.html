SHAZ Player

SHAZ Player was made by me as a side project. At start it was made simply to run multiple formats of videos in multiple windows at a time and to gain more control over frame rate and other aspects of vedios. It turned out to be a complete database backed player with capability to stream vedios from file systems using p2p architecture. SHAZ Player is made from VLCJ - A library provided by the creators of VLC Player and hence uses same underlaying technology with some modifications to bend it according to our needs/requirements.
Category: Desktop Application, Side-Project
Technology Stack: Java SE, MySQL, WebSockets
External Libraries: VLCj
